# PC Monitor

Полупрозрачный оверлей с метриками системы для Windows: CPU, RAM, GPU, сеть и диски. Окно без рамки, поверх всех приложений, клики проходят сквозь него.

## Возможности

- Общая нагрузка CPU и мини-бары по ядрам (до 16)
- RAM, GPU (через GPUtil / nvidia-smi), сеть, диски
- Иконка в системном трее с меню
- Автозапуск с Windows (переключатель в меню трея)

## Зависимости

```bash
pip install psutil pywin32 Pillow pystray GPUtil
```

`GPUtil` опционален — без него секция GPU показывает «—».

## Запуск

```bash
pythonw monitor.pyw
```

Или соберите exe:

```bash
pyinstaller monitor.spec
```

## Трей и автозапуск

После запуска в трее появляется иконка с тремя полосками (CPU-стиль). ПКМ по иконке:

| Пункт | Действие |
|-------|----------|
| **Запускать с Windows** | Вкл/выкл автозапуск через реестр `HKCU\...\Run` |
| **Перемещение** | Вкл/выкл режим перетаскивания оверлея |
| **Выход** | Закрыть приложение |

В обычном режиме окно прозрачно для кликов и закреплено поверх всех окон (через Win32 `HWND_TOPMOST`, периодически восстанавливается). В режиме **Перемещение** появляется жёлтая рамка — перетащите окно мышью. Позиция сохраняется в реестре и восстанавливается при следующем запуске.

При включённом автозапуске из исходников запускается `pythonw monitor.pyw`; для собранного exe — сам `monitor.exe`.

Повторный запуск игнорируется — в трее всегда одна иконка.

## Расположение окна

Оверлей показывается в правом нижнем углу **второго монитора** (если один — на нём).

## Файлы

| Файл | Назначение |
|------|------------|
| `monitor.py` | Исходник (с консолью при `python monitor.py`) |
| `monitor.pyw` | Запуск без консоли, используется PyInstaller |
| `monitor.spec` | Сборка в `monitor.exe` |
| `monitor.vbs` | Устаревший ручной лаунчер (автозапуск — через трей) |
